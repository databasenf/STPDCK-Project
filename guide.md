# 安装指南

## 前提条件
- 需要一台支持浏览器的设备
- GitHub Pages 已部署

## 安装步骤
1. 克隆仓库
    ```bash
    git clone https://github.com/databasenf/STPDCK-Project.git
    ```
2. 打开 `index.html`

---

# 使用方法
1. 打开页面 [STPDCK 项目](https://databasenf.github.io/STPDCK-Project/)
2. 点击导航菜单浏览教程
